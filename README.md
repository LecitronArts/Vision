# Vision

基于 Minecraft 的 1.20.4 PVP 客户端

## 开发须知

 - 若您的编译器可使用内存大小 < 1024Mb ，请尽量设置到 1024Mb 或以上大小，防止编译器以 OutOfMemory 等错误停止构建。


### 开发目标

- [x] 内置账户管理器
- [x] 内置Baritone
- [x] 内置ViaVersion
- [x] ModernUI支持
- [ ] 客户端基础功能
- [ ] bypasses
